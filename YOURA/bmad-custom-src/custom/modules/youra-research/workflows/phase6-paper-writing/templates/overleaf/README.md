# Overleaf Project Structure

> ICML 2025 Paper Template - Auto-generated by Anonymous Research Pipeline

## Quick Start

1. **Download ICML 2025 Style File**
   - Go to: https://icml.cc/Conferences/2025/StyleAuthorInstructions
   - Download `icml2025.sty` and `icml2025.bst`
   - Place in this folder

2. **Upload to Overleaf**
   - Create new Overleaf project
   - Upload this entire folder
   - Set `main.tex` as the main document

3. **Compile**
   - Use pdfLaTeX compiler
   - Run BibTeX for references
   - Compile twice for cross-references
