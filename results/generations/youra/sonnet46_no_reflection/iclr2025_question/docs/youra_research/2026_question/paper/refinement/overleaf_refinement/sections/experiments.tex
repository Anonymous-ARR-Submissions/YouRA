
The experimental setup is described fully in Section 3. Key parameters are summarized here for reference.

\begin{itemize}
\item \textbf{Model:} meta-llama/Meta-Llama-3-8B, bfloat16, single H100 NVL GPU
\item \textbf{Datasets:} TriviaQA rc.nocontext validation split (500 samples); NaturalQuestions open-domain validation split (500 samples)
\item \textbf{Sampling:} N=10 samples per query, temperature=1.0, top_p=0.9, max 50 new tokens
\item \textbf{Entailment model:} microsoft/deberta-large-mnli
\item \textbf{Evaluation:} AUROC with 1000-resample 95\% bootstrap CI
\item \textbf{Conda environment:} youra-h-e1 (Python 3.10)
\end{itemize}
