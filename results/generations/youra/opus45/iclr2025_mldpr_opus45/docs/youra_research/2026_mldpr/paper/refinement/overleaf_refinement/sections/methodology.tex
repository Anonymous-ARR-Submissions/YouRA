
\subsubsection{3.1 Data Collection and Preprocessing}

Time series data were collected from the HuggingFace Hub. Due to API limitations (historical download time series are not available), proxy time series data from the helenqu/astro-time-series dataset (astronomical lightcurve measurements) were used to validate the methodology.

\textbf{Inclusion criteria:}
\begin{itemize}
\item Minimum history: 12 months (sufficient temporal depth)
\item Minimum activity: 100 total observations
\item Data quality: <10\% missing observations
\end{itemize}

\textbf{Preprocessing pipeline:}
\begin{enumerate}
\item Log transformation: \texttt{log(1 + x)} to handle exponential patterns
\item Z-score normalization: standardize to zero mean and unit variance
\item Linear interpolation for missing values (<10\% per series)
\end{enumerate}

\subsubsection{3.2 Level 1: PELT Changepoint Detection}

The first analysis level identifies discrete phases using the PELT algorithm \citep{Killick2012Optimal}.

Given time series \texttt{X = \{x\textbackslash \_1, ..., x\textbackslash \_n\}}, PELT identifies changepoint set \texttt{C = \{c\textbackslash \_1, ..., c\textbackslash \_k\}} minimizing:

\begin{verbatim}
sum_{i=0}^{k} [Cost(X_{c_i+1:c_{i+1}}) + beta]
\end{verbatim}

where \texttt{Cost()} measures within-segment deviation and \texttt{beta} is a penalty preventing overfitting.

\textbf{Configuration:}
\begin{itemize}
\item Penalty: BIC = \texttt{2 log(n)}
\item Cost function: L2 norm
\item Minimum segment length: 3 observations
\end{itemize}

\subsubsection{3.3 Level 2: DTW Trajectory Clustering}

The second level groups trajectories by shape similarity using DTW-based clustering.

DTW aligns two time series X and Y by finding the optimal warping path minimizing cumulative distance:

\begin{verbatim}
DTW(X, Y) = min_W sum_{(i,j) in W} d(x_i, y_j)
\end{verbatim}

\textbf{Clustering configuration:}
\begin{itemize}
\item Algorithm: TimeSeriesKMeans from tslearn v0.6.0
\item Metric: DTW
\item k range evaluated: [3, 8]
\item Selection criterion: silhouette score
\end{itemize}

\textbf{Shape descriptors computed for each trajectory:}
\begin{itemize}
\item Growth ratio: \texttt{(max - min) / mean}
\item Changepoint count: number of PELT-detected transitions
\item Derivative variance: variance of first differences
\item Peak timing: relative position of maximum value
\end{itemize}

\subsubsection{3.4 Stability Validation}

Clustering stability was validated using bootstrap Jaccard analysis:
\begin{enumerate}
\item Resample 80\% of time series with replacement (100 iterations)
\item Recluster each bootstrap sample
\item Compute Jaccard similarity between original and bootstrap assignments
\item Report mean Jaccard across iterations
\end{enumerate}

Threshold: Jaccard > 0.65 indicates robust clustering.
