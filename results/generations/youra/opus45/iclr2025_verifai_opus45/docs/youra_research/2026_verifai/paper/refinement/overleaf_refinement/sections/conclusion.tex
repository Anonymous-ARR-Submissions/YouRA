
This study compared five levels of error feedback granularity for LLM code repair using CodeLlama-7B-Instruct on 304 MBPP runtime error cases. The main findings are:

\begin{enumerate}
\item Granularity has a statistically significant effect on repair success (ANOVA F=23.89, p=3.5e-19, eta-squared=0.059).
\end{enumerate}

\begin{enumerate}
\item For this model and task, minimal feedback (G0: 41.8\%, G1: 40.8\%) yields higher repair success than detailed feedback (G2: 18.4\%, G3: 16.8\%, G4: 22.7\%).
\end{enumerate}

\begin{enumerate}
\item The initial hypotheses predicting optimal intermediate granularity were not supported. The data show G0 outperforming G3 by 25 percentage points.
\end{enumerate}

\begin{enumerate}
\item These results are specific to CodeLlama-7B-Instruct on MBPP and should not be extrapolated to larger models or different tasks without further empirical validation.
\end{enumerate}

Future work should investigate whether the observed pattern holds for larger models, different prompt templates, and more complex debugging tasks.
