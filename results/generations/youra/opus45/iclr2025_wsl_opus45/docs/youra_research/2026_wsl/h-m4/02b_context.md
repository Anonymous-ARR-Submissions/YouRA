## Hypothesis Information

### Statement
Under controlled conditions, if we analyze Grassmann distances per layer type, then some layers (attention vs MLP) will show stronger task-similarity clustering than others (at least one layer type with Cohen's d > 0.8), because different layers encode different aspects of task-specific transformations.

### Type
MECHANISM (Exploratory)

### Rationale
Previous analysis suggested layer-wise variation in clustering strength. Identifying which layers show strongest signal informs practical applications (e.g., layer-selective adapter retrieval) and deepens mechanistic understanding.

## Verification Protocol

### Conceptual Test
Separate Grassmann distance computation by layer type, compute Cohen's d per layer type for within vs between cluster comparison, rank layers by effect size, and test for attention vs MLP systematic differences.

### Success Criteria
- Primary: At least one layer type shows Cohen's d > 0.8
- Secondary: Systematic attention vs MLP difference identified

### Variables (if applicable)
- **Independent Variable:** Layer Type (q_proj, k_proj, v_proj, o_proj vs up_proj, down_proj, gate_proj)
- **Dependent Variable:** Layer-wise Cohen's d for within vs between cluster comparison
- **Controlled Variables:** Base model (TinyLlama-1.1B), LoRA config (r=32, alpha=64), seed variants
