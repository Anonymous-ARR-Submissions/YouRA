## Core Module APIs

### A-2: Seed Tester Module

Applied: PyTorch official determinism protocol

#### setup_determinism(seed: int) -> None
- Seeds all RNG sources (torch, cuda, numpy)
- Enables deterministic CUDA operations
- Sets environment variables for CUBLAS

#### initialize_model_with_seed(architecture: str, seed: int) -> nn.Module
- Applies determinism with specific seed
- Creates model using h-e1 factory
- Returns model with seed-controlled weights

**Tensor Shapes:**
- 1layer: fc1[128,784], fc1.bias[128], fc2[10,128], fc2.bias[10] → ~196K params
- 2layer: fc1[256,784], fc2[128,256], fc3[10,128] + biases → ~400K params

### A-3: Distance Computation

Applied: Euclidean distance metric, combinatorial pairwise comparison

#### compute_pairwise_distances(models_dict: Dict[int, nn.Module]) -> np.ndarray
- Input: 30 models (keyed by seed)
- Process: Flatten parameters, compute L2 norm for all pairs
- Output: [435] distances (30 choose 2)

---

### A-4: Statistical Testing

Applied: One-sample t-test (scipy.stats)

#### test_independence(distances: np.ndarray) -> Dict[str, float]
- Test: H0: mean=0 vs H1: mean>0
- Returns: mean, std, t-stat, p-value, n_pairs
- Success: p < 0.05

---

### A-5: Multi-Condition Runner

Applied: 2×2 factorial design

#### run_all_conditions(config) -> Dict[str, Dict]
- Conditions: 2 architectures × 2 datasets = 4 total
- Each condition: 30 seeds → 435 distances → stats
- Gate: ALL 4 must have p < 0.05
