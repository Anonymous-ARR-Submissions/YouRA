Purpose: experiment workspace for the 2026-03-18 seed-variance study on simple MLPs, organized by hypothesis branches `h-e1`, `h-m1`, `h-m2`, and `h-m3`. Tech stack: Python, PyTorch, torchvision, NumPy, pandas, matplotlib. Structure: each hypothesis folder contains planning docs plus `code/`, and some branches also have top-level `results/` or `figures/`. `h-e1` measures variance on MNIST/Fashion-MNIST, `h-m1` tests seed independence from initialization, `h-m2` analyzes trajectory divergence using saved training artifacts, and `h-m3` runs bootstrap CI stability on `h-e1` outputs.
