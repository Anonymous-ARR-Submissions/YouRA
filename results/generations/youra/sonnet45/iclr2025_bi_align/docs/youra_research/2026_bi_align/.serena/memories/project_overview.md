Research session project for bidirectional alignment via linguistic agency markers in the Anthropic HH-RLHF dataset. Codebase contains two main experiment modules: `h-e1` for extraction feasibility on a 10k sampled response set and `h-m-integrated` for full paired construct validation on 169,352 HH-RLHF preference pairs. Python stack with HuggingFace `datasets`, spaCy, NumPy, pandas, SciPy, matplotlib, and seaborn. Top-level docs/paper artifacts live under `paper/`, experiment code under `h-e1/code` and `h-m-integrated/code`, and result artifacts under `h-e1/`, `h-m-integrated/results`, and `h-m-integrated/figures`.
