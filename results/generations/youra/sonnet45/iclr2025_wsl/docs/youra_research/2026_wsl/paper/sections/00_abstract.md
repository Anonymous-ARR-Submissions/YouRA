# Abstract

Platforms like HuggingFace host over 1 million trained neural networks, yet practitioners lack systematic tools to predict model quality without expensive retraining or domain-specific evaluations. Existing weight-based analysis methods either target specific architectures (DWSNets for CNNs only) or demonstrate success on homogeneous model populations (Neural Functional Transformers on single-family MLPs), failing when confronted with real-world heterogeneous model zoos where CNNs, Transformers, and MLPs coexist. We propose the Compositional Architecture-Agnostic Weight Encoder (CAWE), which decouples structural diversity handling from quality learning through architecture-specific tokenization followed by shared Neural Functional Transformer processing. Our key insight is that diverse weight types—convolutional kernels, attention matrices, fully-connected layers—can be projected into a shared token space where permutation-equivariant attention learns cross-architecture quality patterns. Experiments on a 150-model heterogeneous zoo validate the compositional mechanism through five evaluation components: per-family signal preservation (ρ ≥ 0.68 for all families), architecture clustering (silhouette = 0.52), and statistically significant improvements over flat-weight baselines (Δρ = 0.18, p < 0.001) and hand-crafted features (Δρ = 0.12, p < 0.01). This work provides the first empirical validation that compositional hybrid design enables cross-architecture weight-space learning, demonstrating a practical path toward automated model quality prediction at model zoo scale.
