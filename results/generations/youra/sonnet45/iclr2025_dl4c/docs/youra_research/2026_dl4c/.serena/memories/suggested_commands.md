Useful commands:
- `python h-e1/code/run_mock_experiment.py` to generate simulated H-E1 signatures, metrics, and figures.
- `python h-e1/code/run_experiment_poc.py` to run the minimal real H-E1 experiment on HumanEval+.
- `python h-m-integrated/code/run_analysis.py` to load H-E1 results and generate downstream mechanism-analysis outputs.
- `pytest h-m-integrated/code/tests` for the available automated tests in the integrated analysis layer.
- Standard Linux navigation/search tools observed in use: `ls`, `find`, `rg`, `sed`, `tail`, `head`.
