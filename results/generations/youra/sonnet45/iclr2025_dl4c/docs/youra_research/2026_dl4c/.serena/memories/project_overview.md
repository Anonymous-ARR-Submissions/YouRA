Project purpose: experiment workspace for a DL4C paper on detecting alignment-method signatures in code generation. Main active experiment areas are `h-e1/` (clustering/signature extraction over HumanEval+ or simulated mock signatures) and `h-m-integrated/` (mechanism analysis over saved H-E1 results). There is also `paper/` for manuscript artifacts and `_archive/` for historical recovery snapshots. Tech stack: Python with numpy, pandas, torch, Hugging Face `datasets`, matplotlib/seaborn, and pytest for some analysis tests.
