Project purpose: research workspace for the 20260318_scsl experiment studying geometric structure of spurious features in self-supervised learning embeddings, centered on Waterbirds and the SCSL/clusterability paper pipeline. Tech stack: Python 3.10, PyTorch, torchvision, scikit-learn, numpy, pandas, scipy, matplotlib, pytest, plus Markdown/LaTeX paper assets. Rough structure: h-e1/ contains the base Waterbirds SSL + cluster-balanced retraining code; h-m-integrated/ contains the integrated mechanism-validation and LA-SSL pipeline with checkpoints/results/figures; paper/ contains figure generation and manuscript assets; .data_cache/ holds dataset artifacts; _archive/ contains older recovery snapshots.
