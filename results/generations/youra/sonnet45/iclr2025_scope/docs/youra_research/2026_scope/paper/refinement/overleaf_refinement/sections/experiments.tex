
\subsubsection{Research Questions}

\textbf{RQ1 (Methodology Validation):} Can SVD-based effective rank computation reliably measure intrinsic dimensionality of Transformer projection weights at 7B scale?

\textbf{RQ2 (Low-Rank Structure):} Do deep Transformer layers (L $\geq$ 20) in pre-trained 7B models exhibit effective rank r\_eff < 256?

\textbf{RQ3 (Entropy Decrease):} Does operator entropy decrease monotonically with layer depth ($\beta$ < 0, p < 0.01)?

\subsubsection{Model and Layers}

\textbf{Model:} Mistral-7B-v0.1
\begin{itemize}
\item Architecture: 32-layer decoder-only Transformer
\item Parameters: 7B total
\item Hidden dimension: 4096
\item Attention heads: 32
\item Target layers: 20-31 (deep layers L $\geq$ 20)
\end{itemize}

\subsubsection{Analysis Protocol}

This is a measurement study, not a training experiment. The protocol:

\begin{enumerate}
\item Load pre-trained Mistral-7B (frozen weights)
\item For each layer 20-31:
\begin{itemize}
\item Extract Q, K, V projection weight matrices
\item Compute SVD
\item Calculate effective rank at $\tau$ = 0.99
\item Compute operator entropy
\item Statistical analysis:
\item Linear regression: H(L) vs. L
\item Test: $\beta$ < 0, p < 0.01
\end{itemize}
\end{enumerate}

\subsubsection{Success Criteria}

\textbf{Hypothesis support requires ALL criteria:}
\begin{enumerate}
\item r\_eff < 256 for all layers L $\geq$ 20
\item Entropy slope $\beta$ < 0 with p < 0.01
\item Stable measurements across layers
\end{enumerate}

\subsubsection{Limitations}

\begin{enumerate}
\item \textbf{Single model scale (7B):} Results specific to 7B parameters; may differ at other scales
\item \textbf{Weight analysis, not runtime:} Measures projection weight ranks, not runtime attention matrix (QK^T) ranks
\item \textbf{Architecture specificity:} LLaMA-family decoder-only Transformers only
\item \textbf{Variance threshold:} $\tau$ = 0.99 is standard but arbitrary; other thresholds yield different r\_eff values with same qualitative findings
\end{enumerate}
