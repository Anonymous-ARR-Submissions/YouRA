
\subsubsection{4.1 Model}

CodeLlama-7B base model (not instruction-tuned) from HuggingFace (codellama/CodeLlama-7b-hf). The base model variant was selected to test feedback routing mechanisms without confounding effects from instruction-following training.

\subsubsection{4.2 Hardware}

Single NVIDIA H100 GPU with 95GB memory. Experiments used FP16 precision and CUDA\_VISIBLE\_DEVICES=1 for isolated GPU execution.

\subsubsection{4.3 Dataset}

HumanEval benchmark (164 hand-written Python programming problems) accessed via evalplus package. Test suites use HumanEval+ augmented tests (80+ tests per problem for robustness validation).

\subsubsection{4.4 Reproducibility}

Fixed random seed (42) for all sampling operations. Model weights, dataset version, and library versions documented in requirements.txt files within experiment directories.
