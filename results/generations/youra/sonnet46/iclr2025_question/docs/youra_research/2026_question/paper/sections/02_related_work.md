# Related Work

## Hallucination Detection: Generation-Based Approaches

The dominant paradigm for hallucination detection leverages the LLM's own generation process. SelfCheckGPT [Manakul et al., 2023] samples multiple stochastic outputs and measures consistency via NLI, BERTScore, or n-gram overlap — reasoning that hallucinated content will be inconsistently reproduced across samples. This approach requires no reference data and achieves strong performance on biography generation (AUROC ≈ 0.83), but demands multiple LLM inference passes at detection time, multiplying computational cost by the sample count.

Verbalized confidence methods prompt LLMs to express uncertainty directly [Kadavath et al., 2022]. While instruction-tuned models can partially calibrate this signal, verbalized confidence is unreliable for open-ended generation and inaccessible when the target model is not instruction-tuned [Xiong et al., 2024]. Chain-of-thought verification [Wei et al., 2022] and self-consistency checking [Wang et al., 2023] similarly depend on eliciting multi-step reasoning from the model under evaluation, combining detection overhead with sensitivity to prompt design.

Our approach occupies the opposite extreme: no LLM generation at detection time, no model access beyond a frozen NLI classifier. This generation-free property makes detection cost constant and independent of the monitored model.

## NLI-Based Factual Consistency

The use of NLI models for factual consistency evaluation has a well-established lineage. Early work [Maynez et al., 2020] demonstrated that faithfulness in abstractive summarization could be partially predicted by NLI entailment between the summary and source document. SummaC [Laban et al., 2022] systematized this insight, showing that sentence-level NLI aggregation (SummaCConv) achieves 74.4% balanced accuracy on summarization consistency benchmarks — substantially outperforming document-level NLI. The key design choice is max-pooling over sentence-pair scores, which isolates locally inconsistent sentences rather than diluting the signal with neutral content.

TRUE [Honovich et al., 2022] extended this evaluation to a broader cross-task setting, meta-evaluating NLI-based factual consistency metrics across 11 datasets spanning dialogue, summarization, and QA. TRUE established that DeBERTa-NLI generalizes across task types and that the entailment signal is a reliable factual consistency indicator. Critically, TRUE's evaluation uses balanced accuracy and correlation with human judgments, not AUROC against hallucination labels — leaving the direct hallucination detection performance on benchmarks like HaluEval unmeasured.

More recently, FActScore [Min et al., 2023] proposed atomic-fact decomposition: breaking responses into individual atomic claims, then verifying each claim against a retrieved knowledge source using NLI. FActScore achieves high precision for long-form biographical generation but requires retrieval infrastructure and is computationally expensive per response. ORION [Gerner et al., 2025] demonstrated that post-hoc NLI encoders achieve F1 = 0.83 on RAGTruth hallucination detection without task-specific training, validating the generation-free NLI paradigm in a retrieval-augmented setting.

Our work differs from this line in a critical respect: we evaluate frozen NLI *directly on HaluEval* using pre-existing (context, response, label) triples, without retrieval augmentation or task-specific fine-tuning. This provides the first AUROC measurement for this configuration and enables direct comparison with generation-based baselines on the same benchmark.

## The HaluEval Benchmark

HaluEval [Li et al., 2023] provides large-scale hallucination evaluation data across three task types: dialogue (20,000 balanced pairs), QA (20,000 pairs), and summarization (20,000 pairs). Each sample consists of a grounding context, a generated response, and a binary hallucination label. HaluEval hallucination examples were generated by prompting GPT-3.5 to produce plausible-but-incorrect responses, creating a controlled benchmark for systematic evaluation.

Prior work using HaluEval focused on LLM-based detection methods: ChatGPT-based discriminators, P(True) verbalized confidence, and chain-of-thought reasoning. No prior work has applied frozen discriminative NLI models to HaluEval and measured AUROC — the metric most relevant for ranking-based detection tasks. This gap is what our work fills.

## Hallucination Taxonomy

The distinction between commission and omission hallucinations has been noted informally in prior work. Maynez et al. [2020] categorized summarization errors as intrinsic (contradicts source) vs. extrinsic (not supported by source), corresponding closely to the commission/omission boundary we articulate. SummaC's success on summarization — inconsistency detection achieving 74.4% accuracy — reflects performance on explicitly *inconsistent* (commission-type) summaries in the AggreFact benchmark. HaluEval-Summarization, by contrast, contains GPT-3.5-generated hallucinations that are predominantly omissions or paraphrastic reductions, not direct contradictions.

Our work provides the first quantitative characterization of this boundary using AUROC and mechanistic statistics (Wilcoxon rank-sum, Cohen's d, KL divergence), unifying the observation under a principled theoretical framework applicable to method selection for future hallucination detection systems.
