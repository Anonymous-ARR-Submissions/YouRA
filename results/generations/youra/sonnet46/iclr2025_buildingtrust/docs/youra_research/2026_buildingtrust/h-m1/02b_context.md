## Hypothesis Statement

Under greedy-decoded log-probability evaluation on MMLU, Pythia base models (1.4B, 2.8B, 6.9B) show ECE < 0.15 before any alignment training, confirming that pretraining yields naturally calibrated logit distributions as the causal baseline for alignment-induced shifts.

## Rationale

This hypothesis establishes "Causal Step 1" of the mechanism: that base pretraining yields calibrated outputs from which alignment deviates. If base models are already miscalibrated, alignment-induced ΔECE cannot be attributed to alignment alone. This is a BUILD_ON claim per Phase 2A but requires explicit quantitative confirmation for mechanistic validity in this specific model family.
