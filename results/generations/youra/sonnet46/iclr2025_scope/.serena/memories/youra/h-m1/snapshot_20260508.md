  - Field name: `text` (not `sentence`)
- MNLI: Use `nyu-mll/multi_nli` (NOT `nyu-mll/glue mnli`)
  - Fields: `premise` + `hypothesis`, split: `validation_matched`
- Alpaca: `tatsu-lab/alpaca` ✅
- WikiText: `wikitext/wikitext-103-raw-v1` ✅

### Pingouin API (v0.6.1)
- ICC type name: `"ICC(C,k)"` (NOT `"ICC3k"`)
- CI column: `"CI95"` (NOT `"CI95%"`)

### CUDA Environment Fix
- `conda run -n env python` does NOT pass CUDA visibility
- Use: `env -i HOME=... CUDA_VISIBLE_DEVICES=0 /path/to/env/bin/python script.py`
- Set TRANSFORMERS_OFFLINE=1 HF_DATASETS_OFFLINE=1 when all data cached
