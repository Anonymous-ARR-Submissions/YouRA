
The experiment was conducted on a single NVIDIA H100 NVL GPU. Training used PyTorch 2.0+ with PyTorch Geometric for graph batching. The model was trained for 33 epochs before early stopping, converging to a loss of 0.693 (approximately log(2)), which corresponds to the theoretical minimum for balanced binary classification.

\textbf{Training Configuration:}
\begin{itemize}
\item Optimizer: Adam (lr=10⁻⁴, weight decay=10⁻⁸)
\item Batch size: 32
\item Learning rate schedule: ReduceLROnPlateau (factor=0.5, patience=10)
\item Maximum epochs: 100 (stopped at epoch 33)
\item Random seed: 123
\end{itemize}

\textbf{Evaluation Procedure:}
\begin{enumerate}
\item Load best checkpoint (lowest validation loss)
\item For each of 8 test instances, generate assignment and compute metrics
\item Calculate distribution statistics and compare to gate thresholds
\end{enumerate}
