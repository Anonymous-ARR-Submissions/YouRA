**Type:** EXISTENCE

## Hypothesis Statement

Under epistemic uncertainty conditions (TruthfulQA factual questions), if we extract geometric features (participation ratio, eigenvalue spectrum, condition number) from layers 24-31 hidden states at pre-generation final token position, then these features will achieve Spearman |ρ| > 0.4 correlation with ground-truth semantic entropy, because uncertain model states exhibit measurable geometric signatures.

### Dataset
- **Name:** TruthfulQA (standard)
- **Type:** standard
- **Source:** HuggingFace datasets (truthful_qa/generation)
- **Size:** 817 questions
- **Split:** 70/30 train/test (572/245)
- **Purpose:** Epistemic uncertainty benchmark - questions where models lack factual knowledge

### Controlled Variables

- **SE Parameters:** K=10 samples, T=0.7, DeBERTa-v3-base-mnli-fever-anli for NLI clustering

---

## Variables

### Independent Variables (IV)
- Participation Ratio (PR): trace(C)²/(||C||²_F·d)
- Eigenvalue Decay Rate (α): slope of log-eigenvalue spectrum
- Condition Number (κ): ratio of largest to smallest eigenvalue

### Dependent Variable (DV)
- Semantic Entropy (SE): 0-3 bits from K=10 sampling + NLI clustering

### Controlled Variables (CV)
- Model: Llama-3-8B-Instruct
- Dataset: TruthfulQA
- Layers: 24-31
- Position: final token
- SE computation: K=10, T=0.7, DeBERTa-NLI

---

## Success Criteria

### Primary
- Spearman |ρ| > 0.4 with 95% CI excluding 0.3
- p < 0.001 (highly significant)

### Secondary
- Bootstrap CV < 0.15 for participation ratio (statistical stability)
- Correlation remains significant after controlling for confounds (sequence length, perplexity)

---

## Verification Protocol

1. Load TruthfulQA dataset (817 questions), split 70/30 train/test
2. Extract hidden states from layers 24-31 at final token position for each question
3. Compute geometric features: PR, eigenvalue decay α, condition number κ
4. Compute ground truth SE via K=10 sampling + DeBERTa-NLI clustering
5. Calculate Spearman correlation between each geometric feature and SE on test set
6. Bootstrap validation (1000 resamples) for 95% CI and stability check

## Gate Logic

**Type:** MUST_WORK

**Pass Condition:**
- Spearman |ρ| > 0.4, p < 0.001, 95% CI excludes 0.3

**Fail Action:**
- If |ρ| < 0.3 or p > 0.05: ABANDON hypothesis
- Reason: Geometric properties don't proxy uncertainty, fundamental assumption invalid
